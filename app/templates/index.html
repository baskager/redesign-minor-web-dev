Minor Web Development

This minor teaches you to make interactive applications with HTML, CSS and JavaScript. In separate courses and projects you'll learn everything about: real-time web, internet of things, performance, rapid prototyping...

Sign up!

Courses

Here's a short overview of all the main courses that we teach in the minor web development.

CSS to the rescue

Creating fun, pleasurable and accessible solutions for different kinds of interfaces. Using CSS without the use of classes. Getting to know the basic CSS principles is key!

Go to CSS to the rescue

Web app from scratch

Creating websites is getting more and more complex and is usually not a one person job. It is important to ensure that design is consistent and optimized to meet business objectives and create enjoyable experiences for users.

Go to web apps from scratch

Browser technologies

Here we create robust websites for the web through progressive enhancement. Make sure the basic functionalities work on on as many devices as possible.

Go to browser technologies

Performance matters

The web is getting bigger and faster. Here we're going to make use of caching and service workers, to keep your website accessible and fast for everyone.

Go to performance matters

Real time web

Creating a web app that makes use of sockets to create a live stream of data. You're going to use this data to make it accessible for every user.

Go to real time web

Web Design

Here we're going to make use of rapid prototyping. It is key to create a lot of different prototypes in a short time, since we are going to do a lot of user testing with your prototypes.

Web Design
