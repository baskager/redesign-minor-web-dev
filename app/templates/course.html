Learn goals

  • Student kan zelfstandig een (Node) webapp bouwen met een backend die gebruikt maakt van templating, routing en een remote database.
  • Student kan een live applicatie opleveren waar interactie mee mogelijk is die invloed heeft op de inhoud van de applicatie via manipulatie van de database.
  • Student kan live data vertalen naar een reactive view (datavisualisatie mag) op een manier die voor een eindgebruiker zinnig is.

More about this course

During this course you'll learn to use Socket.io to establish a connection between the browser and the server. You'll combine your knowledge from other courses and your newly aqquired skills from this course to create a progressive web app that retrieves live data from an API, and display the data to the user.
